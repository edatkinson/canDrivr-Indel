CanDrivR-Indel

By adapting DrivR-Base, a state-of-the-art feature annotation tool to annotate INDELs, CanDrivR-Indel is able to breach pre-existing model's accuracy threshold

CanDrivR-Indel can be used by researchers to predict the cancerous pathogenicity of short (<20 base) Insertion Deletion variants (INDELs)

How to use CanDrivR-Indel

1. Upload a file containing the INDELs you wish to predict the pathogenicity of

2. Click the "Predict" button

3. Download the results
